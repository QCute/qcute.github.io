求职网站发布日期

2020.03.07

现在招聘网站几乎都不展示职位的原始发布日期，很多发布了好几个月甚至是好几年的职位，放着用来吸引简历，极大的浪费了求职者的时间、精力和感情。

个人推荐BOSS直聘，无效职位相对较少，主动询问大多数有回复，而其他网站大多数已读不回
而高端职位则推荐猎聘，无论是HR还是猎头，职位匹配度、专业程度和回复效率都比较高

// 查找数据
const scripts = Array
    // 找全部script
    .from(document.querySelectorAll('script'))
    // 筛选类型为application/ld+json
    .filter((s) => s.type == "application/ld+json")
    // 提取script内容
    .map((s) => s.innerText);

// 转JSON
const data = JSON.parse(scripts[0]);
