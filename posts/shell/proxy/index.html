重拾写作的乐趣

一个简单的双向代理

2020.03.10

1. 使用fifo和nc

# client -> proxy:8974 -> server:8998
rm -f /tmp/fifo && mkfifo /tmp/fifo && nc -lkp 8974  < /tmp/fifo  | nc -lkp 8998 > /tmp/fifo

2. 使用socat

# client -> proxy:8974 -> server:8998
socat TCP4-LISTEN:8974,reuseaddr,fork TCP4:127.0.0.1:8998
